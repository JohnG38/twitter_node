# TWITTER COPY with Nodejs

## Description
Ceci est une copie de l'application Twitter faite en NodeJS

## Stack
- NodejS avec framework Express
- Pug comme moteur de template
- MongoDB pour la persistance des données
- Mongoose comme ORM


## Fonctionnalites
- Creation de compte utilisateur
- Ecriture de tweets
- Like, partage, commentaire de tweets
- Suppression de tweets
